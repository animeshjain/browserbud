Settings

What can BrowserBud do?

BrowserBud sees what you see. It knows which page you're on and can read its content. Just ask about it.

Works on any page

Browse to any website - article, docs, forum, whatever. BrowserBud reads the page content so you can ask questions about it: "What does this page say about X?", "Explain this code snippet."

Select text, ask about it

Highlight any text on a page. BrowserBud picks up your selection automatically, so you can ask "What does this mean?" or "Rewrite this" without copy-pasting.

YouTube: video-aware

On YouTube, it goes deeper. It pulls transcripts, knows your current timestamp, and can read comments. Ask "Summarize this video" or "What did the speaker just say?" and it figures out the rest.

YouTube: capture a frame

Click the camera icon on the video player to grab the current frame. You can then ask about what's on screen - diagrams, slides, code, anything visual.
